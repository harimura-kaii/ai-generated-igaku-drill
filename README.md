# 医学の基礎問ドリル（AI生成・非公式デモ）

解剖生理を中心とした5択問題の演習Webアプリです。**問題と解説はAIが作成しています。**

> **⚠️ 最初にお読みください**
>
> - これは個人が学習のために作った**非公式のデモ版**です。教育機関・医療機関とは一切関係ありません。
> - 問題も解説も**AIが作成**しており、**誤りが必ず含まれます**。正確性は保証しません。
> - **医療行為・診断・治療・服薬などの判断には、絶対に使用しないでください。** 医学的助言ではありません。
> - 国家試験の傾向や合否については何も保証しません。

リポジトリ名に `ai-generated` と入れているのは、URLだけが独り歩きしたときにも
「中身はAI生成である」と伝わるようにするためです。

## このサイトの中心は、問題ではなく「悪問報告」の仕組みです

AIで量産した問題バンクは、どれだけ確認しても一定の割合で悪問——事実の誤り、複数正解になりうる
曖昧な選択肢、不明瞭な設問——を含みます。これは運用でゼロにできる種類の欠陥ではないので、
**「誤りは残っている」という前提のほうを設計に組み込みました。**

すべての問題に報告欄があり、悪問を6分類＋該当箇所＋自由記述で報告できます。報告は問題の正準データに
還流し、修正・再ビルドされて次の版に反映されます。つまりこのサイトは、**使われて指摘されるほど
確実に良くなる**ように作られています。

解いていて「この問題はおかしい」と感じたら、それはあなたが正しい可能性が十分あります。
遠慮なく報告してください。それがこのサイトの想定している使い方です。

### 報告の6分類

| # | 分類 |
|---|---|
| ① | 事実誤り（解説や正解が誤っている） |
| ② | 選択肢が曖昧で複数正解になりうる |
| ③ | 設問文が不明瞭 |
| ④ | 解説が不適切（該当箇所で対象を指定） |
| ⑤ | 日本語用語が不適切 |
| ⑥ | 既存の問題集・過去問と酷似している（版権の懸念） |

分類に加えて「該当箇所」（設問文／解説全体／選択肢A〜E）を指定でき、⑥では似ている出典も書けます。
どの箇所への指摘かが分かるほど、修正が正確になります。

⑥の報告を受けても、**その場で問題を自動的に取り下げることはしません**。「似ている気がする」の一言で
出題を止められる仕組みは荒らしに弱く、また医学知識の問題は同じ事実を問う以上、構造が似るのは
自然だからです。報告は人間が確認したうえで対応します。

## 主な機能

- **範囲選択**: 単元をまたいだ分類ツリーから出題範囲を選択（折りたたみ式・範囲ごとの進捗表示つき）
- **出題設定**: 難易度（基礎/標準/応用/国試）、回答状況、出題数、シャッフル、選択肢ごとの解説の表示切替
- **採点モード**: 1問ずつ答え合わせ／最後にまとめて採点（レビュー画面つき）
- **制限時間**: なし／1問ごと／セット全体
- **苦手分野の割り出し**: リーフ（復習できる最小単位）ごとの正答率集計
- **進捗コード**: 学習記録を短い文字列にして別の端末へ持ち運べます（後述）

## 進捗コード（「ふっかつのじゅもん」方式）

このサイトはサーバーを持たない静的サイトで、学習記録はブラウザのlocalStorageにのみ保存されます。
そのままでは別の端末に引き継げないため、回答記録を短い文字列にエンコードして持ち運べるようにしました。

- 記録するのは「どの問題に、どの選択肢で答えたか」だけです
- チェックサムを持たせているので、書き間違えたコードは弾かれます
- **本人確認の仕組みではありません。** 自己申告レベルの識別に過ぎず、工夫すれば書き換えも可能です。
  試験の記録や成績証明としては使わないでください

## 技術構成

- 依存ライブラリなし（Vanilla JS）。ビルド不要、静的ホスティングのみで動作します
- サーバーを持たないため、成績・報告はすべて利用者の端末内（localStorage）に保存されます
- 悪問報告のみ、設定されていればGoogleフォームへ送信されます（`report-config.js`）
